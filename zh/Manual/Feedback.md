# 反馈方式<Badge type="danger" text="反馈" />

## 反馈问题格式
1. 问题具体情况，以及自己尝试做了什么；
2. 你的电脑系统 + `MaaYuan` 版本 + 模拟器版本；
3. log日志压缩包和完整页面截图。

### 获取log日志
1. 关闭 `MaaYuan` ，打开 `MaaYuan` 文件夹，将现有的debug文件夹完整删除掉；
2. 打开 `MaaYuan` ，复现一遍需要反馈的情况；
3. 任务结束后，点击 `MaaYuan` 主页面里的日志栏【导出日志】按钮，确认保持位置后点击保存；
4. 即可获得一个log_日期压缩包，成功导出log日志啦。

### 获取完整页面截图
1. 使用电脑截图而不是手机拍屏！
2. 截图方式：QQ截图、电脑系统自带截图、键盘截图键截图、下载截图软件截图任选其一；
3. 完整页面截图需要包含：模拟器界面 + `MaaYuan` 主窗口（任务列表、日志和任务选项都能完整显示）
4. 如果无法在一张页面获取模拟器及 `MaaYuan` 的内容，那就分开截取，两个截图内容必须完整。

## 反馈至

### 反馈至`收集表格`
  1. 点开[此问卷连接](https://td3nqr3477.feishu.cn/share/base/form/shrcn1zPtP9CZZT35erTCarjEOg)，按照表格内容填写即可；
  2. 记得留下你的ID，我们解决完成会在频道/QQ群艾特你哦，或者问题需要跟进也要联系上你。

### 反馈至`频道-发帖区`
  1. 频道[【加入我们-一键加入】](../Manual/JoinUs.html#%E4%B8%80%E9%94%AE%E5%8A%A0%E5%85%A5)-帖子广场-广陵王互助&MAC互助区（限MAC系统）；
  2. 根据反馈问题的格式发布相关内容，可简单描述，后续再补充详细内容。
  4. 越详细的内容描述会助力大家更好的帮上你。

### 反馈至`QQ群`
  1. QQ群[【加入我们-一键加入】](../Manual/JoinUs.html#%E4%B8%80%E9%94%AE%E5%8A%A0%E5%85%A5)内发起求助；
  2. 根据反馈问题的格式发布相关内容即可
  3. 如果是出现了任务无法运行等物体一定一定一定要附上log日志哦，便于程序佬排查问题。

### 反馈至`AI`
  1. 打开deepseek(或其他AI软件)；
  2. 发送log日志压缩包里的maa文件，以及简短明了的问题描述；
  3. 最后提问：根据以上日志信息，帮我分析可能是什么方面的问题，并告诉我如何解决；
  4. 尝试根据AI的回答解决问题。
