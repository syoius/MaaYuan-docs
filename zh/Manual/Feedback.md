# 反馈方式<Badge type="danger" text="反馈" />

## 导出日志的方式

1. 关闭 `MaaYuan` ，打开 `MaaYuan` 文件夹，将现有的debug文件夹完整删除掉；
2. 打开 `MaaYuan` ，复现一遍；
3. 关闭 `MaaYuan` 并打开新生成的debug文件夹；
4. 找到文件夹内的maa.log右键将其压缩；
5. 可以将这个压缩包发给佬来解决问题啦。

## 反馈至

### 反馈至`收集表格`
  1. 需要准备文字描述问题 + 问题截图（完整无遮挡的显示模拟器及MaaYuan主窗口）；
  2. 点开[此问卷连接](https://td3nqr3477.feishu.cn/share/base/form/shrcn1zPtP9CZZT35erTCarjEOg)，即可填写；
  3. 记得留下你的ID，我们解决完成会在频道艾特你哦，或者问题需要跟进也要联系上你。

### 反馈至`频道-发帖区`
  1. 找到频道-帖子广场-广陵王互助&MAC互助区（限MAC系统）；
  2. 发布咨询你的问题或反馈，附上文字描述问题 + 问题截图；
  3. 文字描述=问题具体情况及你想要获取什么帮助；
  4. 问题截图=完整无遮挡的显示模拟器及MaaYuan主窗口。
  5. 频道加入方式请查看[【加入我们-一键加入】](../Manual/JoinUs.html#%E4%B8%80%E9%94%AE%E5%8A%A0%E5%85%A5)

### 反馈至`QQ群`
  1. 首先需要加入QQ群，QQ群加入方式请查看[【加入我们-一键加入】](../Manual/JoinUs.html#%E4%B8%80%E9%94%AE%E5%8A%A0%E5%85%A5)；
  2. 将问题具体情况+完整页面截图+log日志+MaaYuan版本+电脑环境+自己尝试做了什么（可选）发在群内；
  3. 或求助麻圆救急。

### 反馈至`AI`
  1. 打开deepseek(或其他AI软件)；
  2. 发送maa.log文件，提问：根据以上日志信息，帮我分析可能是什么方面的问题，并告诉我如何解决；
  3. 尝试根据AI的回答解决问题。
