## 当前活动

| 活动名     | 版本提示       |
| ---------- | -------------- |
| 魂生又一串 | `v2.1.1-Beta7及以上` |

支持`自动进入活动`、`选择男主店铺`、`选择遇到踢馆/评选赛是否继续`、开始营业与结业流程。
【烧烤模式】：纯挂机 / 自助烤串。自助烤串支持分别设置【菜单（食材/饮品）】与【烧烤时长】等参数。

**任务说明：**

- 【烧烤店铺选择】：由于会切换到对应男主店铺，注意在对应男主店铺选好菜再开始任务，否则会进入失败；
- 【自动选菜】：进入烧烤前的选菜，可能因文字识别等原因出现问题，可以第一次让麻圆选菜后取消勾选；
- 踢馆模式说明：
  【无脑挂机】直接点开始踢馆并等待失败，继续烤串；
  【手动踢馆后继续】输出日志等待手动踢馆，踢馆完成后继续烤串；
  【直接结束任务】遇到踢馆直接停止整个`MaaYuan`任务；
- 评选赛是否继续：
  【继续】：放弃报名，继续挂机烤串（适用于长时间挂机）；
  【不继续】：直接结束`MaaYuan`任务，适用于后期刷评选赛奖杯；

**常见问题：**

1. 日志提示”未找到xx饮料”：调大【饮品识别延时】，单位ms；
2. 饮料容易空杯直接交付：调大【倒水识别延时】，单位ms；
3. 游戏科技树缩短了时间：调整【xx烧烤时长】（现已按时间长短顺序排序，单位ms）+【饮品倾倒时长】（长按倒水按钮的时间，单位s）
