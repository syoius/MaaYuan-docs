## 当前活动

| 活动名     | 版本提示       |
| ---------- | -------------- |
| 去去指哀牢 | `v2.1.2-Beta2及以上` |
| 夜以继日的绩效评估 | `v2.1.2-Beta1及以上` |
| 披荆斩棘的实习考核 | `v0.9.7及以上` |

**去去指哀牢：**
自动导航、自动清体力，四种辅助模式，兼容 推剧情➡️刷家具➡️卡1心神刷代币 及 0心神无限刷成就。
- 刷家具➡️卡1心神：
把体力消耗到个位数（推剧情/扫荡，etc.）后再使用，可选家具刷齐后是否继续刷登山杖；
- 推剧情➡️刷家具➡️卡1心神刷代币及0心神无限刷：
测试中，暂无适配21层之后的，可能卡随机对话/困难关或者其他鸟的小巧思，建议人工监察，可选家具刷齐后是否继续刷登山杖；
- 卡1心神：
适合已经关卡全通+家具全收集的选手。由于中后期关卡有2波怪，所以会直接导航到第8关进行战斗；
- 无限刷（成就）
适合无视心力反复刷关做密探成就。由于中后期关卡有2波怪，所以会直接导航到第8关进行战斗，可选【游戏内置自动战斗】&【抄作业】需要先在侧边栏的【自动作业】加载/激活好要用的作业。

**夜以继日/披荆斩棘：**
支持`自动答题`、`扫荡`、`自动完成六属性战斗任务`。
请确保完成前置任务（25次鸢报、消耗五铢钱60w、送礼30次、消耗白金币200、消耗家具派遣体力100、打造5家具、合成30次材料、6种属性的战斗、消耗15行囊）后再开始执行本任务！否则未完成的任务会被直接跳过/结束当前任务。
如启用`自动完成六属性战斗任务`请【保证编队的前2位成员能够速刷6-24】并【预留20体力】，启用本功能时会在清活动任务之前自动替换不同属性的队员刷2次6-24。
