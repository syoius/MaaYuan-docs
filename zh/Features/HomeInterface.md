## **自动化任务**

可**设置**一键启动模拟器及开始运行`MaaYuan`任务，并完成任务后自动关闭模拟器和`MaaYuan`

## **任务列表**

| 操作                  | 说明                                     |
| :-------------------- | :--------------------------------------- |
| 全选 / 全不选         | 一键勾选或取消所有任务                   |
| 添加任务              | 预置任务不足时手动补充，也可查找目标任务 |
| 重置任务              | 恢复为内置预设，会覆盖当前预设内容       |
| 修改任务名            | 在任务名旁空白处右键                     |
| 单独运行 / 从此处运行 | 便于局部调试任务流程                     |
| 删除任务              | 直接从当前实例移除                       |
| 跳过任务              | 右键任务名后，当前轮运行会跳过该任务     |
| 任务排序              | 拖动任务即可调整顺序                     |
