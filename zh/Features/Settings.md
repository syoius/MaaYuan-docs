# 设置界面<Badge type="warning" text="设置" />

## 定时执行

可设定定时时间，需保证`MaaYuan`为开启状态，到达时间后会自动执行对应实例中勾选的任务。每个实例所对应的定时执行是独立的，如需要定时执行不同任务，请新建多个实例并勾选所需任务。
- 强制定时启动：勾选后会重启模拟器并开始任务，需在 “设置-启动设置” 中填写好模拟器的启动路径等，详见[【启动设置】](../Features/Settings.html#%E5%90%AF%E5%8A%A8%E8%AE%BE%E7%BD%AE)。

## 性能设置

GPU加速开关，可自行调整。

## 运行设置

- 实时视图：实时监控模拟器视图开关，开启后会在主任务界面及控制台界面监控到模拟器实时画面，会占用一部分运存，可自行调整。
- 实时视图最高刷新率：监控视图刷新率设置，自行调整。
其余项如非必要不建议调整。

## 连接设置

- 捕获方式&触控模式：`MaaYuan`截图及点击的方式，如遇到点击/运行问题，可更换其他选项尝试。
- 记住连接：开启后会记录当前连接的模拟器信息，推荐开启。
- 连接失败后相关设置：根据自身情况调整，如需要`MaaYuan`在多开模式下启动模拟器，请开启“ADB连接失败时尝试启动模拟器”功能，详见[【启动设置】](../Features/Settings.html#%E5%90%AF%E5%8A%A8%E8%AE%BE%E7%BD%AE)。

## 启动设置

可设置模拟器路径，设置完成后，`MaaYuan`将在特定情况下（如开启“启动时操作”/“ADB连接失败时尝试启动模拟器”功能）打开模拟器。
- 启动时&结束后操作：`MaaYuan`启动及任务结束后的操作，根据自身情况调整。（注意：该部分设置仅在单个实例模式下生效，有多个实例的情况下无效）
- 模拟器路径：填写/点击输入框右侧的打开按钮，选择模拟器安装路径中的启动文件，如MuMu模拟器为“MuMuNxMain.exe”（TIP：可选桌面快捷方式）。
- 附加命令：多开模拟器时填写，如不清楚填写规则，可点击“多开快捷设置”进行选择。
- 等待模拟器启动时间：自行填写，默认等待60s。

## 界面设置

可调整`MaaYuan`的界面等个性化操作，包括：

- 背景图片设置
- 应用程序显示语言
- 界面字体大小缩放
- 最小化设置
- 通知开关

## 外部通知

支持任务完成后发送“任务已全部完成”或自定义内容的通知。

### 钉钉

- 电脑端钉钉建群并添加自定义机器人：群设置 → 机器人 → 添加机器人 → 自定义 → 添加。
- 安全设置选“加签”，复制记录密钥和 webhook 地址。
- 打开`MaaYuan`→ 设置 → 外部通知 → 开启钉钉通知，将密钥和地址分别粘贴至对应位置：
- 机器人 Token = webhook 地址
- 钉钉 Secret = 密钥
- 可选填自定义内容，点击“发送测试”验证。

### 邮箱（以 QQ 邮箱为例）

- 登录邮箱网页端，进入设置 → 账号与安全 → POP3/IMAP/SMTP/Exchange/CardDAV 服务，开启服务并生成授权码（不同邮箱步骤类似）。
- 复制授权码，粘贴至`MaaYuan`指定位置，点击“发送测试”接收验证通知。
- 机器人 Token = 你的 QQ 邮箱账号
- 邮箱 Secret = 生成的授权码
